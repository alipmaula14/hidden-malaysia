The mission

More than the postcards

Malaysia's tourism story is dominated by a handful of icons — KLCC, Penang's hawker streets, Langkawi beaches. But the country is far more varied than the postcards suggest. Hidden Malaysia is built to surface what's actually there beyond the well-trodden routes.

We focus on six states most travel guides skim over: Perlis, Kedah, Perak, Pahang, Negeri Sembilan, and Johor. From limestone caves in tin-mining country to a Minangkabau palace built without a single nail, every story here is a place a traveller wouldn't find by accident.

This is a student project — built by final-year Information Technology students at Universiti Tun Hussein Onn Malaysia for our Web Development course (BIT21503). The site uses Node.js, Express, MySQL, and a hand-built interactive SVG map. Everything you see is original work, with stories sourced from on-the-ground travel reporting and properly credited.

How we built this site

Behind the scenes

Hidden Malaysia is a full-stack web application built entirely from scratch — no website builders, no templates, no CSS frameworks. The frontend is plain HTML, CSS, and vanilla JavaScript that talks to a Node.js + Express API, which in turn reads from a MySQL database. The centrepiece is a hand-built interactive SVG map of Peninsular Malaysia where each state is clickable.

Frontend

This project is built with respect for the local communities, photographers, and travel writers whose work informed our content. All images used are properly licensed under Creative Commons or sourced from open repositories. Source attribution for each post's imagery is noted on the individual post pages.

We log search queries (without identifying users) to improve site content. IP addresses are hashed before storage.
